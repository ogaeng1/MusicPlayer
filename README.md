# MusicPlayer (TypeScript + React)

실행 방법

```
1. git clone https://github.com/ogaeng1/MusicPlayer.git
2. cd folderName
3. npm i
4. yarn start
```

## 제공하는 기능

1. 재생, 일시정지, 다음 곡, 이전 곡
2. 프로그레스 바 원하는 부분 클릭시 해당 시점으로 건너뛰기
3. 플레이 리스트 목록(현재 재생중인 노래 표시)

### ※ 음악 파일을 업로드해서 사용하는게 아닙니다.

### (미리 등록된 음악 파일을 듣는 기능입니다)
